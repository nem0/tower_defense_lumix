# Tower Defense Lumix

A tower defense game project built using the Lumix game engine.

## Description

This is a tower defense game featuring various towers, enemies, and particle effects. The game is scripted in Lua and includes UI elements and 3D models.

## Screenshots

![Tower Defense Game Screenshot](screenshot.png)

## Features

- Multiple tower types
- Enemy waves with different behaviors
- Particle effects
- UI sprites for buttons and arrows
- 3D models for tiles and other assets

## Getting Started

To run this project, you need the Lumix engine installed on your system.

1. Download and install Lumix from the official website: [Lumix Engine](https://github.com/nem0/LumixEngine/)
2. Open the folder in Lumix Studio.
3. Run the project from within the editor.

## Project Structure

- `tower_defense.lua`: Main game script, mostly vibe-coded
- `models/`: Contains 3D models and textures (from Kenney)
- `ui/`: UI sprites and elements (from Kenney)
- `probes/`: Lighting probes
